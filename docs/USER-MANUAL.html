1. Introduction

The EDMS manages official documents, files, workflows, and memos. Hierarchy: Ministry ‚Üí Department ‚Üí Section.

  • Dashboard ‚Äî Role-based home: Officer (My Files, Drafts, Memos), Chief Officer/Director/Minister (Pending Approvals), Records Officer (Legal Hold, Archived), Auditor (Audit Logs)
  • Drive ‚Äî Upload files, organize in folders, promote to official documents
  • EDMS ‚Äî Official records with approval workflows
  • Memos ‚Äî Upward, downward, or personal
  • Search ‚Äî Find files and documents

Dashboards: System Admin and Department Admin see Admin dashboards at /admin. All roles see the main Dashboard at /dashboard with role-specific content.
